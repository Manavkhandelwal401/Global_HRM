# Payroll Module

This directory contains the user interface and components for the Payroll & Compliance module.

## Specification Details
- **Detailed Payslip**: Earnings (basic, HRA, allowance), Deductions (PF, Income Tax, ESI), and Employer Contributions are computed and displayed.
- **Documents & Tax declarations**: Form 16 / Tax Declaration downloadable worksheets.
- **Compliance Dashboard**: Tracks Provident Fund (PF), ESI, and Labour Welfare Fund (LWF) contributions.

## Location
- Access via navigation sidebar/bottom bar under **Payroll**.
- URL: `/payroll`
